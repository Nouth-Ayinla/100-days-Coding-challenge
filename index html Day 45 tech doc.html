HTML Basics

HTML stands for HyperText Markup Language.

It provides the structure of web pages.

Tags are used to define elements like headings and paragraphs.

HTML files have a `.html` extension.

Most elements have opening and closing tags.

<p>This is a paragraph</p>
  • <h1> to <h6> for headings
  • <p> for paragraphs
CSS Syntax

CSS stands for Cascading Style Sheets.

It defines how HTML elements are displayed.

Rules consist of selectors and declaration blocks.

CSS can be written inline, internally, or externally.

Classes and IDs are used to target elements.

body { background-color: #fff; }
  • Selectors
  • Properties
  • Values
JavaScript Basics

JavaScript adds interactivity to web pages.

It is a scripting language that runs in the browser.

Variables, functions, and loops are fundamental concepts.

Use the `script` tag to include JavaScript.

JS can also be included from external files.

let x = 5;
  • Variables
  • Functions
  • Events
Best Practices

Use semantic HTML whenever possible.

Write clean and readable code.

Keep CSS and JavaScript in separate files.

Test your site in different browsers.

Optimize performance and accessibility.

function greet(name) { return "Hello " + name; }
  • Use comments
  • Keep it simple
  • Follow standards
